# Security Policy

## Supported Versions

We actively provide security updates for the latest release of Photo Scanner.

| Version | Supported |
| ------- | --------- |
| v0.8.x  | ✅ Yes    |
| v0.7.x  | ❌ No     |
| < v0.7  | ❌ No     |

## Reporting a Vulnerability

We take the security of your local photo data very seriously. If you discover a vulnerability, please report it privately.

**Contact**: [jarjav2005@gmail.com](mailto:jarjav2005@gmail.com)

Please include:
1. A description of the vulnerability.
2. Steps to reproduce the issue.
3. Potential impact on user data or photo library integrity.

We will acknowledge your report within 48 hours and provide a timeline for resolution. **Please do not disclose the vulnerability publicly until we have provided a fix.**

*Thank you for helping keep Photo Scanner secure.*
